Although I no longer have time to work on games, in the past I've worked on making small interactive games. These games were created using Macromedia Flash 8 (now called Adobe Animate) and programmed via Actionscript 2.0 which is based on Javascript. In the future, I hope to learn either mobile or PC game development, given the rise in mobile-friendly gaming engines such as Unity and console-optimized gaming engine Unreal.

Cave World
An avoider game with high-score API implementation, "cave room", playable here. The player is a green square that must avoid the falling spikes (triangles) and get the green, blue and yellow squares. The green squares extend the players health bar, the blue squares add points, and the yellow squares slow down the time.



Project Movement
This game was developed as the "main" game. In it, the player is an orange sphere that can move between multiple areas. There are obstacles to get through. Each area had a theme, a forest area, a cave area, a snowy area, and a volcanic area. The game is to date incomplete because the time needed to complete development was too much. Again, a working copy of the game can be provided upon request.



Green Block World
The player is the small green square and has to collect as many blue squares as possible. Meanwhile, a black square moves towards the player. If the that square gets close enough, the health bar starts going down. Keep moving!
